---

## Сводка по уровням себестоимости

| Уровень | Функции | Основные драйверы затрат |
|---------|---------|--------------------------|
| **Низкий** | Сохранения, чтение трендов, сессия/профиль, подборка из БД | Postgres, CPU Vercel |
| **Средний** | «Ещё ролики», транскрибация, ежедневный sync конкурентов | Квота YouTube, Groq Whisper, записи Prisma |
| **Высокий** | Внешний ingest поиска, добавление/обновление конкурентов, сценарии | TikHub, YouTube search units, DeepSeek |

## Таблицы себестоимости по функциям

### Низкая себестоимость

| Функция | Инфраструктура | API | Риск для маржи | Примечания |
|---------|----------------|-----|----------------|------------|
| **Сохранённые ролики** | CRUD `savedVideo`; проверка до 400 ID | Нет | Низкий | Чистый Postgres |
| **Тренды (чтение)** | Poll `GET /api/trends/realtime` | Нет при чтении | Низкий | Клиент опрашивает часто — **счёт Vercel** на масштабе, не TikHub |
| **Профиль / сессия** | `SessionUser`, cookie, `UserTokenBalance` | Нет | Низкий | До API упирается в пул Supabase |
| **Первая подборка ленты (БД)** | `findMany(800)` + фильтр в памяти | Нет | Низкий–средний | Тяжёлый запрос — CPU/БД |
| **Авторизация (NextAuth)** | JWT + Google OAuth | OAuth бесплатно | Низкий | |

**Риск для маржи:** по отдельности низкий. **Опрос трендов вне вкладки Home** (из аудита) умножает холостые read-запросы.

### Средняя себестоимость

| Функция | Инфраструктура | API | Риск для маржи | Примечания |
|---------|----------------|-----|----------------|------------|
| **«Ещё ролики»** (`action: "more"`) | Тот же DB-путь + возможный ingest | YouTube при пустом пуле; TikHub для IG | Средний | 5 токенов в коде; порция = 8 роликов |
| **Транскрибация** | Таймаут маршрута 120 с | Groq Whisper **⚠️ ~$0,001–0,01/мин (~0,09–0,85 ₽/мин)** | Средний | 5 токенов; субтитры обходят API |
| **Ежедневный sync конкурента** | `competitor-daily-sync.ts` | TikHub/YouTube на профиль | Средний | 5 токенов/профиль/действие |
| **Лог поисковых запросов** | `searchQueryLog.create` | Нет | Низкий | Рост хранилища |

**Риск для маржи:** активный «load more» без сценариев всё равно жжёт YouTube/TikHub.

### Высокая себестоимость

| Функция | Инфраструктура | API | Риск для маржи | Примечания |
|---------|----------------|-----|----------------|------------|
| **Внешний ingest поиска** | Sync YT + IG в HTTP | YouTube **100 units/search** ⚠️; TikHub **⚠️** | **Высокий** | Throttle 15 мин/запрос (`search-throttle.ts`) |
| **Добавить конкурента (IG)** | Fetch профиля + persist | TikHub **⚠️** | Высокий | 30 токенов |
| **Обновить Reels конкурента** | Bulk ingest | TikHub **⚠️** | Высокий | 30 токенов |
| **Генерация сценария** | Полная история чата | DeepSeek **⚠️ ~$0,01–0,05/ген (~0,85–4,25 ₽)** | Высокий | 20 токенов; `max_tokens: 1800` |
| **Lazy refresh трендов** | Фоновое наполнение пула | Возможен YT/TikHub | Средний–высокий | `POST /api/trends/lazy-refresh` |

**Риск для маржи:** Visioner с 25k токенов без async ingest может дать **отрицательную валовую маржу** при худшем TikHub — веса токенов калибровать по счетам.

# Токеномика

## Принципы

1. **Токены ≈ COGS** — дорогие действия (TikHub, DeepSeek) стоят больше.
2. **Первый поиск бесплатен** — как в коде; воронка без трения.
3. **Текст для пользователя на русском** — «сколько стоит действие».
4. **Планы дают месячный пул** — см. `pricing-overview.md`; кошелёк в коде только списывает.

## COGS на 1 000 потраченных токенов (⚠️ модель)

Микс Creator: 50% load-more / 30% сценарии / 20% шпион:

| Компонент | ₽ / 1k токенов | USD / 1k (≈) |
|-----------|----------------:|-------------:|
| TikHub + YouTube | **⚠️ 120–350 ₽** | $1,3–3,7 (~110–315 ₽) |
| DeepSeek | **⚠️ 40–100 ₽** | $0,4–1,1 (~34–94 ₽) |
| Инфра (аллокация) | **⚠️ 30–80 ₽** | $0,3–0,8 (~26–68 ₽) |
| **COGS итого** | **⚠️ 190–530 ₽** | $2–5,6 (~170–476 ₽) |

Creator: 2 990 ₽ / 4 000 токенов → **0,75 ₽ выручки на токен**.

**Валовая маржа на потребление токенов:** ~30–75% в зависимости от TikHub — **проверить до масштабирования рекламы**.
