### Цены

| Вопрос | Ответ |
|--------|-------|
| Менять цену? | **Нет** — запуск с Creator 2 990 ₽ / Visioner 14 990 ₽ |
| Средний тариф? | **Нет** — два уровня проще; биллинг не усложнять |
| Годовые? | **Мягкий запуск** 29 900 / 149 900 ₽ после платёжки |

### Лимиты

| Лимит | Действие |
|-------|----------|
| Заменить demo 12 400 | **Да — P0** · привязка к плану при подписке |
| Внешний ingest на Free | **Выкл.** до биллинга |

1. Таблица подписок + webhooks (YooKassa/Stripe)
2. Job начисления токенов по плану
3. `ensureSessionUser` проверяет план перед spend
4. Убрать «оплата позже» из прод-UI

## 100 платящих (~480k ₽ MRR)

### Цены

| Вопрос | Ответ |
|--------|-------|
| Менять цену? | **Держать**, пока COGS TikHub &lt; 35% выручки |
| Средний тариф? | **Оценить**, если &gt;30% тикетов «нужно больше токенов, но не Visioner» |
| Пакеты токенов? | **Запустить** S/M/L (см. `feature-costs.md`) |

### Лимиты

| Лимит | Creator | Visioner | Причина |
|-------|---------|----------|---------|
| Токены / мес | 4 000 | 25 000 | Держать, пока p90 usage &lt; 80% |
| Конкуренты | 5 | 15 | Max 10 в коде — только если COGS ок |
| Перенос | 0 | 0 | Нет накопления перед churn |

### Триггеры подорожания (+10–15%)

- TikHub &gt; **0,50 ₽** на эквивалент load-more
- Подорожание DeepSeek
- Vercel &gt; **15%** MRR

## 500 платящих (~2,4M ₽ MRR)

### Цены

| Вопрос | Ответ |
|--------|-------|
| Менять цену? | **+10% для новых** · grandfather 6 мес |
| Средний тариф? | **Да — если данные** · см. ниже |
| Годовая скидка | Push **20%** (вместо 17%) в lifecycle-письмах |

### Предложенный средний тариф: **Producer — 7 990 ₽ / мес** (условно)

Добавлять **только если**:

- &gt;25% Creator покупают пакеты 2×/мес **ИЛИ**
- В поддержке: «нужно 10 конкурентов», но бюджета на агентство нет

| Параметр | Producer (предложено) |
|----------|---------------------|
| Цена | 7 990 ₽ / мес · 79 900 ₽ / год |
| Токены | 10 000 / мес |
| Конкуренты | 10 |
| Сохранения | 500 |
| Аудитория | Сильный эксперт + мини-команда, SMM на нескольких клиентов |

**Не добавлять**, если апгрейды на Visioner уже забирают heavy users.

### Лимиты

- **Мягкие caps** с авто-докупкой пакета **предложено**
- Ingest в **async queue** — те же цены, лучше маржа

## 1 000 платящих (~4,8M ₽ MRR)

### Цены

| Вопрос | Ответ |
|--------|-------|
| Менять цену? | **Creator → 3 290 ₽** для новых, если маржа &lt; 65% (pessimistic) |
| Visioner | Держать 14 990 ₽ — якорь для агентств |
| Producer | Сохранить, если запущен |

### Лимиты

| Изменение | Причина |
|-----------|---------|
| Visioner 25k → 30k токенов | Perk удержания при churn &gt; 6% |
| Место в команде +3 990 ₽/seat | Новая строка выручки; общие списки конкурентов **предложено** |
| Rate limits API по плану | Защита контракта TikHub |

### Упаковка

- **Без Content Radar** — фокус
- «Приоритетная поддержка» только в Visioner

## 5 000 платящих (~24M ₽ MRR)

### Цены

| Вопрос | Ответ |
|--------|-------|
| Enterprise | **Custom** — SSO, SLA, выделенная квота TikHub |
| Self-serve | Creator **3 490 ₽**; Visioner **16 990 ₽** |
| Free Preview | Ужесточить до **100 токенов** — CAC на масштабе |

### Лимиты

- **Жёсткий дневной бюджет TikHub** по тарифу
- Региональные цены (СНГ vs РФ) — только с локальной оплатой

### Сетка из трёх тарифов

```
Creator 3 490 ₽  →  Producer 8 990 ₽  →  Visioner 16 990 ₽
```

Переименование — по бренд-исследованию; цены ~+15% к запуску.
